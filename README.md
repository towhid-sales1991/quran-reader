# আল-কুরআন রিডার (Al-Quran Reader)

সূরা ও পারা অনুযায়ী আরবি কুরআন (Uthmani script) এবং বাংলা অনুবাদ একসাথে পড়ার জন্য একটি আধুনিক, রেসপন্সিভ ওয়েব অ্যাপ।

## ফিচার

- ১১৪টি সূরা ও ৩০টি পারা — দুই ভাবে ব্রাউজ করা যায়
- নাম বা নম্বর দিয়ে সার্চ
- আরবি (উসমানী স্ক্রিপ্ট) + বাংলা অনুবাদ (একসাথে)
- ফন্ট সাইজ পরিবর্তন ও অনুবাদ দেখানো/লুকানোর সেটিংস
- পূর্বে যেখানে পড়েছেন, সেখান থেকে চালিয়ে যাওয়ার অপশন (লোকাল স্টোরেজ)
- মোবাইল ও ডেস্কটপ — দুই জায়গাতেই ব্যবহার-উপযোগী ডিজাইন

## কন্টেন্ট সোর্স

আরবি ও বাংলা টেক্সট লোড হয় [AlQuran Cloud API](https://alquran.cloud/api) থেকে (ফ্রি, রিয়েল-টাইম, কোনো key লাগে না)। তাই ব্যবহারকারীর ইন্টারনেট সংযোগ প্রয়োজন।

## মোবাইলে অ্যাপ হিসেবে ইনস্টল করা (PWA)

এই ওয়েবসাইটটা একটা **PWA (Progressive Web App)** — মানে এটা মোবাইলে সত্যিকারের অ্যাপের মতো ইনস্টল করা যায়, কোনো Play Store/App Store লাগে না।

**Android (Chrome):**
1. GitHub Pages লিংকটা Chrome-এ খুলুন
2. উপরে ডান পাশে ⋮ মেনুতে ট্যাপ করুন
3. **"Add to Home screen"** বা **"Install app"** সিলেক্ট করুন
4. কনফার্ম করলে হোমস্ক্রিনে আইকন চলে আসবে

**iPhone (Safari):**
1. লিংকটা Safari-এ খুলুন
2. নিচে শেয়ার বাটনে (□↑) ট্যাপ করুন
3. **"Add to Home Screen"** সিলেক্ট করুন
4. **Add** ট্যাপ করুন

ইনস্টল করার পর এটা ফুল-স্ক্রিনে খুলবে, ব্রাউজারের address bar থাকবে না — ঠিক নরমাল অ্যাপের মতো। সূরা/পারার তালিকা অফলাইনেও দেখা যাবে, কিন্তু আয়াত পড়ার জন্য ইন্টারনেট লাগবে (কারণ আরবি ও বাংলা টেক্সট লাইভ API থেকে আসে)।

## GitHub Pages-এ আপলোড করার পদ্ধতি

1. GitHub-এ একটি নতুন repository তৈরি করুন (যেমন `quran-reader`)
2. এই ফোল্ডারের সব ফাইল (`index.html`, `css/`, `js/`) আপলোড করুন
3. Repository **Settings → Pages**-এ যান
4. **Branch: main**, ফোল্ডার **/ (root)** সিলেক্ট করে **Save** করুন
5. কিছুক্ষণ পর `https://<your-username>.github.io/quran-reader/` লিংকে সাইট লাইভ হয়ে যাবে

## ফোল্ডার গঠন

```
index.html          → মূল পেজ (HTML গঠন)
style.css            → সব স্টাইল (থিম সহ)
surahs-data.js       → ১১৪ সূরার তথ্য
juz-data.js          → ৩০ পারার তথ্য
app.js               → মূল লজিক (নেভিগেশন, সার্চ, রিডার, সেটিংস)
manifest.json        → PWA তথ্য (অ্যাপের নাম, আইকন, রঙ)
service-worker.js    → অফলাইন সাপোর্ট (app shell cache)
icon-192.png         → অ্যাপ আইকন (ছোট সাইজ)
icon-512.png         → অ্যাপ আইকন (বড় সাইজ)
```

## লোকালি টেস্ট করতে

ফোল্ডারে গিয়ে একটি সাধারণ সার্ভার চালু করুন:

```bash
python3 -m http.server 8000
```

তারপর ব্রাউজারে `http://localhost:8000` খুলুন।
